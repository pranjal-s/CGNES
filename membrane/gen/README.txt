# gen: initial configuration generators

### Description of files and directories in this directory
README.md           this file
structures.ipynb    development of nm.py
NE.png              screenshot of NE from assembly simulation
                    (NM is to be reverse engineered from this)
hcp.jpg             HCP lattice example
                    (from https://docs.lammps.org/Howto_pylammps.html)
nm_make/            to generate NM structure file (nm.data)

### Usage
1. structures.ipynb is for educational purpose for newcommers.
2. The README in nm_make/ directory shall guide users on creating nm.data required by LAMMPS in sim/.
